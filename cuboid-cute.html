2. 切割位置

值必须在 0 和 长度 之间
值必须在 0 和 宽度 之间
值必须在 0 和 高度 之间

3. 表面积计算

初始表面积:

公式: 2 * (长 * 宽 + 长 * 高 + 宽 * 高)

--

计算结果: --

切割增加面积:

沿长度切割增加: 2 * (长切割数) * (宽 * 高)

--

沿宽度切割增加: 2 * (宽切割数) * (长 * 高)

--

沿高度切割增加: 2 * (高切割数) * (长 * 宽)

--

总增加 = 长增加 + 宽增加 + 高增加

--

计算结果: --

最终总面积:

公式: 初始面积 + 总增加面积
